# OS Banker Simulator

Ứng dụng web mô phỏng Banker's Algorithm dùng trong hệ điều hành để kiểm tra trạng thái an toàn và phòng tránh deadlock. Hỗ trợ nhập ma trận tài nguyên, chạy từng bước mô phỏng và kiểm tra thứ tự thực thi hợp lệ.

## Yêu cầu

- [Node.js](https://nodejs.org/) >= 18
- npm (đi kèm với Node.js)

## Cài đặt và chạy

```bash
# Cài đặt dependencies
npm install

# Chạy ở chế độ development
npm run dev
```

Mở trình duyệt và truy cập `http://localhost:5173`.

## Build production

```bash
npm run build
npm run preview
```
